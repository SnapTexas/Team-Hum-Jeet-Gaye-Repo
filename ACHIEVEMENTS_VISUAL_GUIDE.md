## Real-Time Updates

### Update Triggers:
1. **On App Launch** - Load initial data
2. **Every Step** - Update live step count
3. **Every Minute** - Refresh weekly/monthly stats
4. **On Refresh** - Manual data refresh
5. **On Tab Change** - Load tab-specific data

### Data Refresh Rate:
- **Live Steps**: Real-time (as sensor updates)
- **Calories**: Real-time (calculated from steps)
- **Distance**: Real-time (calculated from steps)
- **Weekly Stats**: Every minute
- **Monthly Stats**: Every minute
- **Badges**: Real-time (calculated from metrics)
